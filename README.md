# 🏦 Bank Management System (Python)

A simple **Bank Management System** developed using **Python**, **Object-Oriented Programming (OOP)**, and **JSON** for data storage. This project allows users to create bank accounts, deposit and withdraw money, update account details, and delete accounts through a command-line interface.

---

## 📌 Features

* ✅ Create a New Bank Account
* 💰 Deposit Money
* 💸 Withdraw Money
* 👤 View Account Details
* ✏️ Update Account Information
* 🗑️ Delete Account
* 🔐 PIN Verification
* 🆔 Automatic Account Number Generation
* 💾 JSON File as Database
* 🧩 Object-Oriented Programming (OOP)

## ⚙️ How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Bank-Management-System.git
```

### 2. Open the Project Folder

```bash
cd Bank-Management-System
```

### 3. Run the Program

```bash
python main.py
```

or (macOS/Linux)

```bash
python3 main.py
```
